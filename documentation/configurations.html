Configurations

Warning

To use any of these commands, you need to have Manage Guild permissions or be in one or more of the admin_roles config roles set by the server.

Administrator Roles​

UsageDescriptionPremium
/config admin_roles [hidden]View the server's admin rolesFalse
/config admin_role add <role>Add a role to the admin rolesFalse
/config admin_role remove <role>Remove a role from the admin rolesFalse

Moderator Roles​

UsageDescriptionPremium
/config moderator_roles [hidden]View the server's moderator rolesFalse
/config moderator_role add <role>Add a role to the moderator rolesFalse
/config moderator_role remove <role>Remove a role from the moderator rolesFalse

Suggestion Configurations​

UsageDescriptionPremium
/config suggestions_channel <#channel> Set the servers suggestion channelFalse
/config discussion_threads <value> Set the whether a thread should be created on suggestion creationFalse
