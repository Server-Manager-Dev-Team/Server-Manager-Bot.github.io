Moderation

UsageDescriptionPremium
/warn <member> [duration][reason] [custom_reason][moderator_hidden]Warn a member (moderator_hidden hides the moderator from the actioned user)False
/warnings [member]False
/remove_warning <id> [notify_user=True]Remove a warning (notify_user will direct message the warned member that it was deleted)False
/delete_all_warningsDelete ALL warnings from your server. This is useful if your bot is acting slow when warning usersTrue
/ban <member> [reason][custom_reason] [moderator_hidden][delete_message_days]Ban a member from the server (delete_message_days is the amount of message days to delete)False
/kick <member> [reason][custom_reason] [moderator_hidden]Kick a member from the serverFalse
/purge <amount>Purge an amount of messages in the current channelFalse
/mute <member> <duration> [reason][custom_reason] [moderator_hidden]Mute a member for a duration using Discord's Timeout Feature. (Example duration argument: '1h' = 1 hour)False
danger

Using the /delete_all_warnings command cannot be undone!

caution

Discord's Timeout feature only allows members to be muted with a max duration of 1 week. If you try and mute someone for over a week, the bot will send you an error message saying you can't do that
