This release includes a new panel system, with limits for premium and non-premium servers.


New commands

Name

Premium

 Usage Description
config panels False /config panels View the panels for the current server
config panels add False /config panels add <name> <channel> <ticket_type> [category] Create a panel (limit: 5 premium, 3 non-premium)
config panels remove False /config panels remove <name> Remove a panel by name - this may be changed to a random ID as some point.

Timed Polls


Timed polls are now implemented into the poll command which adds an duration argument, which is a humanized-readable time format.

Before After
/poll start <topic> [ping_role] [channel] /poll start <topic> [duration] [ping_role] [channel]
