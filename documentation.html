Documentation



Commands marked in red are required.

Commands marked in yellow are optional.

Commands marked with ⭐ are premium commands.


Setup Commands (MANAGE GUILD REQUIRED)

/setup • Setup your server in an interactive proccess

/config channel <#channel> • Set the channel where users will create tickets in

/config category <@category> • Set the category where tickets will be created (MANAGE CHANNEL PERMISSIONS REQUIRED)

/config ticket_embed • Customize the embed that is sent on a ticket being created

/config message • Set the message that is sent on ticket creation (top of the embed)

/config delete_on_leave <value> • Set whether tickets should be deleted when the author of it leaves

/config log_channel <#channel> • Set the log channel for the bot (MANAGE WEBHOOKS REQUIRED)

/config log_events • Set what events are dispatched to the log channel when happening (Ticket Create, Ticket Close, Ticket Delete, Ticket Transcripts)

/config staff_roles <@role> <type: add, reset, remove> • Add a role to the staff roles, remove a role from the staff roles, or reset the staff roles

/config blacklist_roles <@role> <type: add, reset, remove> • Add a role to the blacklist roles, remove a role from the blacklist roles, or reset the blacklist roles (CANNOT CREATE TICKETS)

⭐ /config reviews • Configure reviews on a ticket being deleted/closed


Ticket Commands (PERMISSIONS: CANNOT BE TICKET AUTHOR)

/add <@user> [reason] • Add a user to the ticket

/remove <@user> [reason] • Remove a user from the ticket

/request_close <reason> • Request the ticket author to give you permission to close the ticket


General Commands

/help • Shows the help command

/stats • Shows the statistics of the bot
